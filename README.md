🎓 SmartCampus Management System
📌 Overview

The SmartCampus Management System is a console-based Java application designed to manage students, courses, and their enrollments.

This project demonstrates core Java concepts like OOP, collections, exception handling, and multithreading in a practical scenario.

🎯 Purpose

The goal of this project is to help an admin:

Store and manage student records
Manage course details
Enroll students into courses
Process enrollments efficiently
✨ Key Features
👨‍🎓 Student Module
Add new student
Stores:
Student ID
Name
Email
📘 Course Module
Add course
Stores:
Course ID
Course Name
Course Fee
📝 Enrollment Module
A student can enroll in multiple courses
Fast data access using HashMap
Prevents duplicate enrollments (custom logic)
📋 Menu System

The program runs using a menu:

Add Student
Add Course
Enroll Student
View Students
View Enrollments
Process Enrollment
Exit
⚙️ Technologies & Concepts Used
🔹 Object-Oriented Programming
Classes & Objects
Constructors
Encapsulation
Method Overriding (if extended)
🔹 Collections Framework
ArrayList → store students and courses
HashMap → manage enrollments
🔹 Exception Handling
Handles invalid input (wrong data type, negative values)
Custom Exceptions used:
InvalidFeeException
InvalidStudentException
🔹 Multithreading
Enrollment processing runs in a separate thread
Simulates real-time processing
🔹 File Handling (Optional)
Save and load data from file
💡 Additional Feature (Unique)
Email format validation before adding a student
Prevents invalid user data entry# my-first-project-assigment
